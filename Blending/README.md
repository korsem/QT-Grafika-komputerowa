# Blending
## Program Blending umożliwia blending trzech obrazków w pięciu wybranych trybach z możliwością regulowania intensywności blendingu.
Program umożliwia tryby:
* normal </br> 
* score </br>
* multiply </br>
* difference </br>
* overlay </br>
